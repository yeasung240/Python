```python
def solution(word):
    alphabets = ['A', 'E', 'I', 'O', 'U']
    global answer 
    answer = 0 

    def dfs(string):
        global answer 
        answer += 1 
        #guard closure 
        if string == word:
            return True
        #guard closure 
        if len(string) == 5:
            return False
        #find the given word 
        for alpha in alphabets:
            if dfs(string + alpha) == True:
                return True 

    for alpha in alphabets:
        if dfs(alpha) == True:
            return answer


```

#### 공부한 내용 본인의 언어로 정리하기 ##
DFS 완전 탐색문제.
